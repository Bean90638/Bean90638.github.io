本文分享我開發機上會安裝的工具

1 截圖

  • Snipaste: 截圖好工具

2 版本控管

  • Git: 檔案時光機等級，IT人員必備
  • TortoiseGit: 我都叫他小烏龜🐢，方便Git的GUI操作
  • TortoiseSVN: 這也是小烏龜🐢，SVN偏伺服端安裝的，本機版控時不會考慮使用

3 資料庫

  • SSMS: 專門用於SQL ServerMSSQL的資料庫管理工具
  • DBeaver: 支援多類型資料庫的管理工具

4 IDE

  • Visual Studio: 地表最強開發IDE
  • Visual Studio Code: 俗稱VSCode
  • Cursor: 結合AI的VSCode，試用後覺得很方便，猶豫要不要課金

5 應用程式

  • NotePad++
  • LINQPad
  • Notepad2e
  • 7Zip: 免費的壓縮、解壓縮工具
  • wox: 快速啟動應用程式
  • everything: 電腦內所有檔案建立查詢索引，可以wox配合快速搜尋檔案
  • Postman: 測試API
  • WinMerge: 文字內容差異比對，可比對純文字、檔案、目錄內所有檔案
  • WinSCP: UI介面操作FTP

6 網站

  • Miro: 便條紙白板
  • excalidraw: 手繪風格白板
  • draw.io: 畫流程圖的好工具
  • sequencediagram: 畫時序圖的好工具
  • dbdiagram 資料庫設計網站
  • regex101 驗證正則表達式

7 套件

7.1 Visual Studio

  • GitHub Copilot
    AI文字提示工具，開發時會跳出提示文字，按下Tab就可以套用，但是缺點是要錢 大概一個月20鎂
  • codeium
    免費仔用的AI文字提示工具，開發時會跳出提示文字，按下Tab就可以套用
  • Toggle Comment
    註解快捷工具，按下Ctrl+/可以快速註解或取消註解。
  • SonarLint
    靜態程式碼分析工具，會在有問題的程式碼字串出現綠色波浪，提示快速修改或轉跳說明網站
  • ClaudiaIDE
    更改美美的背景圖，可以每季把新老婆放上來一起Coding(?)
  • MatchMargin
    框選文字時候凸顯其他同樣文字位置
  • Spell Checker
    英文拼字檢查工具，在Visual Studio中檢查拼字是否正確文章有教排除中文錯誤
  • GhostDoc
    自動撰寫XML註解工具，在方法前使用///時候會辨識方法及變數套用到註解上

7.2 VSCode

  • Codeium: AI Coding Autocomplete and Chat for Python, Javascript, Typescript, Java, Go, and mo
    免費的AI文字提示工具，開發時會跳出提示文字，按下Tab就可以套用
  • AWS Toolkit - Amazon Q, CodeWhisperer, and more
    免費的AI文字提示工具，開發時會跳出提示文字，按下Tab就可以套用
  • Encode Decode
    文字編碼轉換工具，像是符號轉換HTML編碼、加密之類的
    預設是Shift+Ctrl+P輸入Encode/Decode: Convert Selection，我是設定Ctrl+Ali+X
  • Git Extension Pack
    保哥精選Git工具大禮包
  • Markdown All in One
    Markdown工具大禮包
  • Code Spell Checker
    英文拼字確認
  • Error Lens
    錯誤訊息於該行後面呈現，但有時會有點亂會關掉

7.3 Edge&Chrome

兩邊瀏覽器的擴充套記基本上互通就一起介紹

  • 新同文堂
    簡繁轉換工具，可以設定自動簡轉繁&轉換字典
  • Chrome Remote Desktop
    神奇遠端程式，綁定Google帳號並輸入對應pin碼，只要有開機就算沒登入也可以直接連入遠端電腦(怕)
  • ChatGPT 萬能工具箱
    ChatGPT提供快捷按鈕、URL帶參數快速查詢，可拿來配合wow工具快速啟動
  • Adblock
    擋廣告(有些整版廣告真的太惱人了，想支持的再開白名單)
  • Video Speed Controller
    影片加速快捷鍵工具
    預設是G2倍速，我會改成 G1.5倍速 H2倍速
  • JSON Formatter
    網頁上的JSON以格式化方式呈現
  • Dark Reader
    強制網頁深色模式保護眼睛
    設定Alt+C加入僅反轉列出清單
  • Tampermonkey
    網頁腳本擴充程式

7.4 Tampermonkey

  • Fix <code> bug for Edge translator: 修正Edge翻譯後標籤文字位移問題
  • 圖片瀏覽器
